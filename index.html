Welcome to my Building highways of Growth project.

Purpose

This website was created to help people who, like me, struggle to take the time to research and learn new skills. This website will teach leadership and interpersonal skills that can help you grow as a person.

Website Structure

This website has a Getting Started page for helping people find where they want to start. It also has 4 learning modules: Leadership Within, leadership Without, Learning how to learn, and Personal Finance. The Getting Started page explains each module.

These modules each have several sections with individual pages for each section.

This website is intended to teach basic principles and concepts rather than pure data.

My sources are linked in the bottom.

Who am I?

Shaun Vann

I'm a student at Brigham Young University-Idaho. I really enjoy learning, but I find it very difficult to take the time to do all the research that's needed to learn something new. Because of this struggle, I have strived to make it easier for other people to learn without having to do the research themselves.
